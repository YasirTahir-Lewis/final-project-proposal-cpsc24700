Application Overview:

This is going to be a web based application open for public use, here user can take a Maths quiz with multiple choice questions. And get to see you many they got right.



Techonology Overview:

I will be using HTML, CSS and JavaScript for the client side programming, I will make sure the website looks appealing, attractive and easy to use for new users. For the web server side I will use JSON for all the logical programming since it goes well with Javascript. And for the database I will use JSON file as a data file.
